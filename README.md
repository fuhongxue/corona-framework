# Corona Framework

MVC Framework for Corona SDK.

## Getting Started

### 1. Submodules

```bash:console
$ git submodule init
$ git submodule update
```

### 2. GameAnalytics

```lua:config/application.lua
GA = {
  GAME_KEY   = "YOUR_GA_GAME_KEY",
  SECRET_KEY = "YOUR_GA_SECRET_KEY",
  BUILD_NAME = "1.0",
}
```
